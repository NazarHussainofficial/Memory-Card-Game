# 🎴 Memory Card Game

An interactive, engaging, and fully responsive memory card match game built as a showcase of modern frontend development skills. 

## 🚀 Features
- **Dynamic Grid:** Beautifully animated cards layout.
- **Smooth Animations:** CSS 3D transforms for realistic card-flipping effects.
- **Game Logic:** Built-in win-detection, score counters, and flip-limit matching logic.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop screens.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+) 
- **Icons/Styles:** Custom custom fonts & styling

## 🔧 How to Play
1. Click on any card to flip it and reveal the symbol/image.
2. Click a second card to try and find a matching pair.
3. If they match, they stay face up. If they don't, they flip back over.
4. Match all the pairs in the fewest moves possible to win!
